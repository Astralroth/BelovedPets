El funcionamiento de este proyecto es simple, los pasos que se deben seguir son:
1.- ejecutar por consola de comandos el archivo c_venv, el cual creara el ambiente virtual de python y lo poblara con los requerimientos.
2.- ejecutar por consola de comandos el archivo init.bat, el cual ejecutara todo.
3.- Disfrutar el proyecto.

requerimientos:
    - Python@X.X instalado en el sistema.
